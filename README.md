# 🌳 SmartTree AI

سیستم هوشمند تشخیص و درمان بیماری‌های گیاهان با استفاده از هوش مصنوعی

## ✨ ویژگی‌ها

- 🤖 تشخیص بیماری با هوش مصنوعی (آپلود تصویر برگ)
- 🌿 ۲۴ بیماری گیاهی با درمان علمی
- 🌍 نقشه جهانی شیوع بیماری‌ها
- 💬 چت‌بات هوشمند
- 📊 داشبورد مدیریت
- 🌓 دارک/لایت مود
- 📞 مشاوره تلفنی: ۰۹۳۸۱۴۹۱۰۵۸

## 🛠️ تکنولوژی‌ها

- HTML5, CSS3, JavaScript
- Chart.js (نمودار)
- Leaflet (نقشه)
- AOS (انیمیشن)
- Font Awesome (آیکون‌ها)
